xt Commerce Connector
=====================

Bundle handles integration of xt:Commerce e-shop database, and supports:

- Full (initial) import of:
    - Categories
    - Content
    - Images
    - Products (simple)
- Sync (partial) import of:
    - Categories
    - Content
    - Images
    - Products (simple)
